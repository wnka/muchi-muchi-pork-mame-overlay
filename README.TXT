     PINK SWEETS 1.0 Mame Layout file

Original art by CAVE Interactive Co., Ltd.
              Design by baf
            Coding by Olifante

               Version 0.9

HOW TO USE:
Drop this zip file in the mame/artwork directory.
(for example, if you installed mame in C:\mame, place the file at C:\mame\artwork\pinkswts.zip).
Enable the Layout helper plugin in mame under "Configure Options" > "Plugins".
Start playing Pink Sweets with the pinkswts.zip romset as usual.
